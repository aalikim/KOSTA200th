## Servlet Attribute and Scope
### Servlet Attribute
> 다음 세 가지 Servlet API 클래스에 저장되는 객체
- ServletContext
- HttpSession
- HttpServletRequest

### Servlet Scope
> Servlet API에 바인딩된 속성에 대한 접근 범위 의미
- 기능
  - 로그인 상태 유지 기능
  - 장바구니 기능
  -  MVC의 Model & View의 데이터 전달 기능
- Scope의 종류와 특징
<table>
    <tr>
        <th>Scope</th>
        <th>Servlet API</th>
        <th>Attribute of Scope</th>
    </tr>
    <tr>
        <td>Application Scope</td>
        <td>ServletContext</td>
        <td>애플리케이션 전체에 대해 접근</td>
    </tr>
    <tr>
        <td>Session Scope</td>
        <td>HttpSession</td>
        <td>브라우저에서만 접근</td>
    </tr>
    <tr>
        <td>Request Scope</td>
        <td>HttpServletRequest</td>
        <td>해당 요청/응답 사이클에서만 접근</td>
    </tr>
</table>
